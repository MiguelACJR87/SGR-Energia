 # SGR-Energia

 Pequeno site estático para relatórios de energia.

 Estrutura principal:

- `index.html` — página principal
- `script.js` — scripts do cliente
- `style.css` — estilos

Como rodar localmente:

1. Abra `index.html` no seu navegador (duplo clique ou `File > Open`).

Publicação (opções):

- GitHub Pages: crie um repositório remoto, faça push e habilite Pages na branch `main`.
- Hospedagem estática (Netlify/Vercel): faça deploy do repositório.

Contato: mantenha este arquivo atualizado com instruções específicas do repositório.
